INGApplePayTransaction

@available(iOS 11.0, *)
public struct INGApplePayTransaction

An object that represents a request for payment, including information about payment processing capabilities, the payment amount, and contact information.

    Parameters

    merchantIdentifier

    Your merchant identifier
    supportedNetworks

    By default is set to [.visa, .mastercard], property constrains the payment methods that the user can select to fund the payment
    supportedCountries

    By default i set to [“PL”], list of ISO 3166 country codes to limit payments to cards from specific countries or regions.
    merchantCapabilities

    By default i set to .capability3DS, A bit field of the payment processing protocols and card types that you support
    countryCode

    By default i set to “PL”, The merchant’s two-letter ISO 3166 country code.
    currencyCode

    By default i set to “PLN”, The three-letter ISO 4217 currency code that determines the currency this payment request uses.
    paymentSummaryItems

    An array of payment summary item objects that summarize the amount of the payment
